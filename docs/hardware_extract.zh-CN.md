## 电气说明

- `BAT_ADC` 使用 `100k / 100k` 分压，因此分压比为 `2.0`。
- `UART0` 专用于 IMU，不能复用于 console 或调试输出。
- `USB CDC` 专用于主机 console、CLI 协议和调试事件流。
- 与摄像头相关的网络在当前固件代中全部忽略。

## 板级规则

- GPIO 定义只能放在 `main/board/board_config.h` 和 `main/board/board_config.c` 中。
- 业务逻辑模块禁止硬编码 GPIO 编号。
- 电机逻辑编号、物理位置和旋向是三个独立概念，不能混用。
